# 数组操作
给定一个长度为 n 的整数数组。

数组中只包含 1 和 −1。

你需要对该数组进行如下操作：

计算该数组中所有元素的和 s。

计算该数组的最小前缀和 x。

输出 s−x 的值。

注意：

数组的最小前缀和 x 有可能为负。

对于任意数组，其前 0 个元素的前缀和为 0。

## 输入格式
第一行包含整数 n。

第二行包含 n 个整数，表示给定数组。

## 输出格式
一个整数，表示 s−x 的值。

## 数据范围
前四个测试点满足 1≤n≤5。

所有测试点满足 1≤n≤100，数组中只包含 1 和 −1。
