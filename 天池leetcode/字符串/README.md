# 字符串简介
简称为串，是由零个或多个字符组成的有限序列

> str = "Hello World"

在上面示例代码中，str是一个字符串的变量名称，Hello World则是该字符串的值，字符串的长度为11.

根据字符串的特点，我们可以将字符串问题分为以下几种：
- 字符串匹配问题
- 子串相关问题
- 前缀/后缀相关问题
- 回文串相关问题
- 子序列相关问题
