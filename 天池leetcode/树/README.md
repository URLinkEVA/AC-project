## 数的定义
 树是N（N>=0）个节点的有限集合。N=0时为空树，N>0时应满足：
1. 有且仅有一个特定的称为根的节点
2. N>1时，其余节点可分为m(m>0)个互不相交的有限集合。其中，每一个有限集合自身也是一棵树。
